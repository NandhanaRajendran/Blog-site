# 📝 Personal Blog Platform

A full-stack blogging platform built using the **MERN Stack (MongoDB, Express.js, React, and Node.js)**. The platform allows users to create, manage, and explore blog posts through a modern and responsive interface. It also includes a dedicated admin dashboard for content management.

> **Status:** 🚧 Under Active Development

---

## 🚀 Features

### User
- Create, edit, and delete blog posts
- Upload images
- Search blogs
- Filter blogs by category
- Like blog posts
- View count tracking
- Responsive design

### Admin
- Separate admin dashboard
- Manage blog posts
- Feature blogs
- Manage users

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/NandhanaRajendran/Blog-site.git
cd Blog-site
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

### Create `.env`

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm run dev
```
